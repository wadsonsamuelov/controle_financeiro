## Visão geral

O FinTrack é uma aplicação web para controle de finanças pessoais. O sistema centraliza receitas, despesas, metas e eventos financeiros em um único painel, oferecendo uma visão clara da situação financeira do usuário sem depender de planilhas.

**Problema:** organizar o controle financeiro pessoal de forma visual, acessível e gratuita.

**Público-alvo:** pessoas que querem acompanhar gastos e receitas de forma simples, sem ferramentas complexas ou pagas.

<br>

## Funcionalidades

| Módulo | Descrição |
|---|---|
| Dashboard | Indicadores financeiros e gráficos (Chart.js) |
| Transações | Cadastro e listagem de receitas e despesas |
| Metas | Criação e acompanhamento de objetivos financeiros |
| Calendário | Eventos e datas financeiras importantes |
| Distribuição | Análise de gastos por categoria e método |
| Câmbio | Cotações em tempo real (USD, EUR, BTC) e conversor de moedas |
| Tema | Modo claro e escuro |
| Persistência | Armazenamento local via `localStorage` |
